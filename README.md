# Drivingo Drive Thru
### Drive Thru application for the customers and restaurant service. ( v2.0.1 )

## Technologies
- React - v18.2.0
- Node.js - v16.20.1
- Material UI
- Tailwindcss
- Mediapipe v0.1.0
- Amazon Web Service

## Configuration
- Node.js v16.20.1
- NPM
- Python

## How to run
- npm run build
- npm run start

Run the client locally. http://localhost:3017
